### ✅ 2. TDD 기반 간단한 문제 해결 (2~3일)
> 이건 우테코 프리코스 스타일에 맞는 연습이야!

- 예제 문제 1: 문자열 계산기  
- 예제 문제 2: 로또 생성기

**학습 포인트:**

- JUnit5 사용법 + 테스트 우선 작성 연습
- 커밋을 기능 단위로 쪼개서 기록하기 (Git 연습)
